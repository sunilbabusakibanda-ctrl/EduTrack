EduTrack Installation Guide

A comprehensive step-by-step manual for the EduTrack School Management System.

1 Enable Person Accounts

Required Prerequisite: Person Accounts must be enabled before installing the package.

  1. Log in to your Salesforce Org as a System Administrator.
  2. Go to Setup.
  3. In the Quick Find box, enter Account Settings.
  4. Check if Allow Customer Support to enable Person Accounts is available.
  5. Proceed to enable Person Accounts via the Enable button.
IMPORTANT: Person Accounts enablement is permanent and cannot be undone. Always verify you are in the correct environment (Production or Sandbox).

2 Package Installation

Once Person Accounts are active, install the EduTrack managed package.

  1. Use the following installation link:
    2. Install EduTrack Package (v1.0.0-1)
  2. Choose Install for All Users.
  3. Click Install.
  4. Wait for the confirmation email from Salesforce.

3 Initial Configuration

Configure the system to reflect your school's branding and operational structure.

3.1 Permissions Assignment

  • Go to Setup > Permission Sets.
  • Select EduTrack Full Access.
  • Click Manage Assignments and assign to relevant users.

3.2 Setup Wizard

  1. Open the EduPro Dashboard app from the App Launcher.
  2. Complete the School Setup form.
  3. Required fields: School Name, Logo, Email, UPI IDs.
  4. Click Save Setup.

3.3 Operational Setup

  • Academic Year: Define the current academic cycle (e.g., 2025-2026).
  • Classes & Sections: Create your primary class structures.
  • Bank Accounts: Add payment collection accounts for automated invoicing.
